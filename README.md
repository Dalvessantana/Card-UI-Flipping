# 📇 DevCard UI

Um cartão de perfil **3D interativo** desenvolvido com **HTML** e **CSS**, com efeito de *flip* (rotação), focado em design moderno, animações suaves e boas práticas de front-end.

---

## 📌 Sobre o Projeto

O **DevCard UI Flipping** é um componente de interface que simula um perfil de usuário com um diferencial visual: um **efeito 3D de flipping**, onde o cartão pode girar revelando informações adicionais.

Esse tipo de interação é muito utilizado em interfaces modernas para melhorar a experiência do usuário e tornar o design mais dinâmico.

---

## 🎯 Objetivo

Este projeto foi criado com o objetivo de:

- Praticar HTML e CSS  
- Aprender e aplicar **transformações 3D (CSS)**  
- Trabalhar com animações e transições suaves  
- Criar interfaces interativas e modernas  

---

## ✨ Funcionalidades

- Efeito **3D flip (frente e verso do card)**  
- Foto de perfil centralizada  
- Nome e profissão  
- Ícones de redes sociais  
- Botões de ação (Contrate / Mensagem)  

---

## 🎨 Design

O design segue princípios modernos:

- Layout limpo  
- Cores equilibradas  
- Bordas arredondadas  
- Animações suaves  
- Experiência visual interativa com efeito 3D  

---

## 🚀 Tecnologias Utilizadas

- HTML5  
- CSS3 (com uso de **transform**, **perspective** e **transition**)
